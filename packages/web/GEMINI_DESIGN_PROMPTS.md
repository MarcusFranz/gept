## Usage Tips

1. **Gemini 2.0 Flash** works well for UI mockups
2. Add "UI design, Figma style, clean vectors" for sharper results
3. Include "dark mode, #1c1c1c background" consistently
4. Reference "gaming aesthetic" and "fintech/trading app" for the right vibe
5. Ask for variations: "Generate 4 variations of this design"
